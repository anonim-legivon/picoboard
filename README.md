# Picoboard
Picoboard - is a modern and is a modern and highly functional engine for anonymous forum (imageboard).

## Features

* Frontend based on React.js
* Tripcodes
* reCAPTCHA 
* Full REST API
* Post markup
* Easy customization
* Fault tolerance

## Technical requirements
To work must be installed this:
* Docker
* docker-compose
* Python (version 3.7+)
* Pipenv
